# Declaration ##

* Syntax,

```
var <pointer_name> *<data_type>
```

* Example,

```
var ptr *string
```



# Initialisation #

* Syntax, 

```
var <pointer_name> *<data_type> = &<variable_name>
```

* Example,

```
i := 10
var ptr *int = &i
```

* Another way of iniialising a pointer is, 

```
var <pointer_name> = &<variable_name>
```

* Example,

```
p := "Programming"
var ptr = &p
```
